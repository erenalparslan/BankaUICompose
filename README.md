# Jetpack Compose Banka Uygulaması

Bu proje, Jetpack Compose kullanılarak geliştirilmiş bir banka uygulaması UI tasarımını içermektedir.

## Özellikler

- Bottom Navigation Bar
- System UI Controller
- Material Extensions
